

                                                                **My Food Project**

                                        I built a website for a fictional premium food delivery service.

##  Technologies I Used
1. **HTML5** 
2. **CSS3** 
3. **Flexbox & CSS Grid** 
4. **Icons** 
5. **Google Fonts** 

## Features of the Website
1. A hero header with a dark background image and call-to-action buttons.
2. A features section with a grid layout.
3. A photo gallery of the food that zooms in when you hover over it.
4. City cards showing where the delivery is available.
5. Customer reviews (testimonials) with a fixed background image.
6. Pricing tables for different subscription plans.
7. **Responsive Design:** I used `@media` queries to try and make the website look good on tablets and mobile phones too!


##  How to view my project
It is super easy to run this project because it is just static web files. You don't need any special software!
1. Download or clone this folder to your computer.
2. Find the `index.html` file inside the folder.
3. Double-click `index.html` and it will open right up in your web browser!
